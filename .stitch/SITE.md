# Sammamish Circle Site Vision

## 1. Vision
A cinematic, dark-themed community hub focusing on goals, resources, and community action in Sammamish. Bringing a premium "Wander.com" style visual feel to local civic engagement.

## 2. Target Audience
Sammamish residents, volunteers, and local organizers looking for high-utility tools in a beautiful interface.

## 3. Core Values
- Premium aesthetics (Cinematic Dark Theme)
- High utility (Functional-First React Components)
- Clear interactions (Marigold Hotspots and CTAs)

## 4. Sitemap
- [ ] index (Homepage)
- [ ] directory (Resource Directory & Interactive Map)
- [ ] dashboard (User Access & Saved Resources)

## 5. Roadmap
- Generate `index` page using Stitch constraints
- Generate `directory` page UI shell
- Generate `dashboard` UI shell
- Surgically inject existing React logic (`useState`, `useEffect`, `localStorage`) into the new Shells.

## 6. Creative Freedom
- Mapbox/Leaflet dark-theme base map styling
- Cinematic spatial drift parallax backgrounds
